# Module 4: How to Prompt AI

AI can help you code, write, and solve problems. The key is knowing how to ask!

## What is a Prompt?
A prompt is what you type to tell the AI what you want.

## Good Prompt Example
> "Write a Python function that adds two numbers."

## Bad Prompt Example
> "Code."

**Tips:**
- Be specific
- Give examples
- State your goal

**Mini Quiz:**
- What makes a good prompt?
- Why be specific?
