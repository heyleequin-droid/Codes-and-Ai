# Module 5: Prompt Engineering Tips & Tricks

Become a prompt pro! Here are some tips:

- **Be clear and direct**: "Summarize this article in 3 sentences."
- **Set the format**: "List the steps as bullet points."
- **Give context**: "Act as a Python tutor."
- **Iterate**: If the result isn't right, rephrase your prompt.

**Prompt Templates:**
- "Explain [concept] like I'm 5."
- "Write a [type] code for [task]."
- "Correct this code: [paste code]"

**Mini Quiz:**
- What does it mean to iterate on a prompt?
- Why give context?
