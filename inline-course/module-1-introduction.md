# Module 1: Introduction

Welcome! In this course, you'll learn the basics of coding and how to prompt AI to get the outcomes you desire.

**Why learn to code?**
- Coding lets you build, automate, and create.
- It helps you think logically and solve problems.

**Why learn to prompt AI?**
- AI can help you code, write, and create faster.
- The better your prompts, the better your results!

**Fun Fact:**
> Coding and prompting are superpowers anyone can learn!
