# List of Useful Prompts

Use these prompts to get the most out of AI:

1. "Explain [concept] in simple terms."
2. "Write a Python function to [do something]."
3. "Debug this code: [paste code]"
4. "Suggest project ideas for beginners."
5. "Quiz me on coding basics."
6. "Summarize this article in 5 bullet points."
7. "Act as a coding tutor and help me learn loops."
8. "Give me step-by-step instructions for [task]."
9. "What are common mistakes in [topic]?"
10. "How can I improve this code: [paste code]"
