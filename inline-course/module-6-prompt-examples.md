# Module 6: Real-World Prompt Examples

Here are some prompts you can use right now:

1. "Write a Python script to sort a list of numbers."
2. "Explain how a for loop works in simple terms."
3. "Generate a quiz about coding basics."
4. "Debug this code: [your code here]"
5. "Suggest project ideas for beginners."

**Mini Quiz:**
- Which prompt would you use to get help with an error?
- How can you ask for examples?
